1. Features

Notes & Folders is yet another notes application. It's not the best one, but only here you can organize your notes by folders just like in real file system.

Current key features are:

  • Plain text notes, checklists and folders
  • Password protection and 128-bit AES encryption
  • Importing UTF-8 *.txt files
  • Searching
  • Backup & Restore

I'm planning to implement these:

  • Exporting data
  • Color schemes
  • Multiple selection for copy/cut/paste

And these features I don't want and won't implement because there're already many powerful applications having them:

  • Images, sounds, videos
  • Tags, links
  • Text formatting
  • Widgets for taking quick notes

I wanted Notes & Folders to be an Android clone of my old MyNotes (which is a similar application for Java ME).

2. Details

2.1. Encryption

Encryption algorithm is AES with 128-bit key.

Your data is encrypted with key that is randomly generated at the first start of application and that never changes. The key itself is encrypted with password that you use to log in. The password in its turn is hashed with SHA-1 algorithm.

Only text content of notes is encrypted. Names, dates and other system information stay not encrypted.

2.2. Data import

It is about importing *.txt files. Their text content is expected to be in UTF-8 encoding. If it's not there would probably be problems with reading notes.

You can choose either a single *.txt file or a folder to import.

The imported data goes into the folder "Imported at YYYY.MM.DD HH:MM:SS" that is created in the root folder. You have to manually copy/move data from it to the place you want.

2.3. Checklists

You can reorder items by drag'n'dropping them (to start dragging an item click it on the right side)

2.4. Backup & Restore

Backup process saves data in a single file on sdcard. The data being saved is:

  • SHA-1 hash of password
  • Key used to encrypted data (encrypted with password)
  • Data (folders, notes and checklists). Only text content is encrypted. Names and dates stay not encrypted.

File is created in /mnt/sdcard/NotesAndFolders folder (it is created automatically if it doesn't exist). File has an *.nf1 extension.

You can use that files to restore data from them. The process of restoring data involves following steps:

  • Deleting all data in database
  • Restoring password and encryption key
  • Restoring data
  • Restarting the application
